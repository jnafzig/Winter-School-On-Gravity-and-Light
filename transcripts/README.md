# Transcripts

Transcripts from Dr. Frederic P. Schuller's [WE-Heraeus International Winter School on Gravity and Light (2015)](https://www.youtube.com/playlist?list=PLFeEvEPtX_0S6vxxiiNPrJbLu9aK1UVC_) YouTube playlist.

- `.txt` — plain text with timestamps
- `.json3` — YouTube subtitle format with timing metadata
- `audio/` — extracted audio and Whisper transcription JSON for lectures without English subtitles

## Source

Most transcripts were downloaded from YouTube English subtitles using `yt-dlp`. Lectures 18–20, 23–25 had no English subtitles available, so audio was extracted and transcribed locally (see `audio/`).
